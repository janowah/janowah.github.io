git

create new directory and copy contents from online repo into it

- make directory and sychronise
  - mkdir janowah
  - cd janowah/
  - git init
  - git remote add origin https://github.com/janowah/janowah.github.io.git ; don't need ssh keys
  - git pull origin master
  - ls -la
- make change to index.html in directory
  - git status ; to see changes not staged
  - git add index.html ; updates what will be committed
  - git diff --staged ; shows what changes have been made
- synchronise
  - git commit -m "your message" ; must add message
  - git push origin master
- points
  - prompted for password everytime you push
- copy changes on remote to local
  - git pull origin master
- setting up ssh keys
  1) generate keys on local machine
     - ssh-keygen -t rsa -b 4096 -C "homer@powerplant.com" ; creates 2 files
  2) add to ssh agent
     - =ssh-add ~/.ssh/id_rsa=
     - check ssh agent =ssh-add -l=
  3) copy keys to github account =cat ~/.ssh/id_rsa.pub=
  4) test ssh access to github
     - ssh -T git@github.com
  5) urls for repo should be git@... and not https
     - git remote set-url origin git@github.com:janowah/janowah.github.io.git
