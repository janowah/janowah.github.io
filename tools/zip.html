zip

NOTES
  - using zip on the command line by default ~ backup files
    weren't archived

CREATE
  - zip -r target source
  - zip -j target source/*  ; leave out top level directory
    -j --junk-paths         ; do not store directory names

EXTRACT
  - unzip -l foo.zip   ; list
  - unzip foo.zip      ; unzip
  - unzip -tq foo.zip  ; test
    -q quiet
    -t test
