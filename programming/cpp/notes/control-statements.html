if

if (expression)
  {statement}

if (x == y)
  {x = x + 5;}

if else

if (expression)
  {if true statement}
else
  {if false statement}

if (x > 5)
  {x++;}
else
  {x = x + 5;}

Conditional operator ?

?: Ternary Condition x = (a < b) ? a : b; right to left 

conditionalExpression ? trueExpression1 : falseExpression2

(x > 5) ? x++ : x = x + 5;

// Display the shorter of s and t
cout << (s.length() < t.length() ? s : t);

Note that the expressions must have the same type.

for (init, comparison, modifier)

switch

switch (int)
  {
    case (expression): statement
    case (expression): statement
    default : (expression)
  }

int x = 5;
switch(x)
{
  case 1: cout << "test1";
    break;
  case 5: cout << "test5";
    break;
  default: cout << "none";
}

break

break quits out of the loop/switch, without completing the remaining statements in the loop/switch.

usage:

most common usage break statement is in some form of infinite loop, such as while(true){},

continue

continue on the other hand continues directly to the next iteration of loop without executing the remaining statements in the loop.
