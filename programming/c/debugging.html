print (‘x’, ‘d’, ‘u’, ‘o’, ‘t’, ‘a’, ‘c’, ‘f’, ‘s’),

Format letters are

  • o(octal)
  • x(hex)
  • d(decimal)
  • u(unsigned decimal)
  • t(binary)
  • f(float)
  • a(address)
  • i(instruction)
  • c(char)
  • s(string)
