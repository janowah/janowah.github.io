recovering root password

boot into single user mode, mount two partitions (/ and /usr) and run passwd

  • put into single user mode
    • interrupt boot
      • boot -s
    • mount partitions
      • fsck -p / && mount -uw /
      • fsck -p /usr && mount /usr
    • change password
      • passwd

https://www.openbsd.org/faq/faq8.html#LostPW
