external css

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="style.css" />

embed css

  1. HTML_HEAD file variables 
    #+HTML_HEAD: <style type="text/css" >
#+HTML_HEAD_EXTRA: h1, h2 {color: #990000;}
#+HTML_HEAD_EXTRA: </style>
  2. embed HTML. this works but puts the style element below the toc. 
    #+BEGIN_HTML
  <style>

  h1, h2 { color: #990000;
  }

  ul {
  list-style-type: none;
  } 
  </style>
#+END_HTML
  3. customise org-html-head
    • useful if exporting several files
