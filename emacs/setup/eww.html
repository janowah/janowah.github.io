=======

  copy these files into the emacs bin directory
  - libxml2-2.dll
  - libiconv-2.dll
  - zlib1.dll

  available from
    http://sourceforge.net/projects/ezwinports/files/

  they are all contained in this file
    libxml2-2.7.8-w32-bin.zip
